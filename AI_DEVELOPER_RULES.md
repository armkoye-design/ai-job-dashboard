# AI Developer Rules

This document contains permanent rules for any AI assistant (Codex or ChatGPT) working on this repository.

## Rules

- Always read PROJECT_GOALS.md before modifying code.
- Never modify unrelated functions.
- Make the smallest possible change.
- Explain every modification.
- Ask questions instead of guessing requirements.
- Preserve backward compatibility.
- Never remove existing features without approval.
- Never change scoring philosophy without approval.
- Never replace a working scraper unless requested.
- Prefer maintainable code.
- Prefer precision over broad modifications.
- Test code before recommending a commit.
